Detailed Description

◆ getCorrectedP2()

QPointF core::RestrictedLineF::getCorrectedP2 ( const QPointF &  p2)

Returns a corrected point P2 (in case of apply). A corrected point is obtained when some restriction apply to this line. If you input some point that is not inside restriction range then it will be processed.

This function is useful when user want to check a point before set it.

Note: correction ONLY apply over P2 because you must remember P1 is the start point of the line so it's not anchored to anything.
