Detailed Description

The DataRepresentation class controls the way the data will be interpreted and shown to the user. Contains all the information of the representation the user wants to use when data is imported from some data source. This class is not graphic, only contains data that could help to get a graphical representation.

Data could be represented in different ways like Images, matrix, sound, text, charts, or simply raw data.

Note
Data won't be changed when this class is instantiated, it just will make a representation of data itself.
