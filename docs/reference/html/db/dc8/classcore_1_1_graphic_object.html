Detailed Description

The GraphicObject class is the base class for any object added to an enviroment or system. This class has all basic methods and functions to make basic operations like cut, copy, paste, and manipulate as an abstract object.

A GraphicObject basicly consist of a rectangle with dots in the corners and sides to manipulate it. Anything could be drawn inside this rectangle.
