Detailed Description

BasicTable es una tabla basica que se puede usar a nivel general en diferentes widgets. Esta tabla tiene funciones basicas como copiar, cortar y pegar. Cada uno de estos metodos estan ya definidos pero pueden ser sobrescritos por las clases que hereden de esta.
