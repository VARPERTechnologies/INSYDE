core::BasicDialog
The BasicDialog class represents an improved and with-basic-functionalities dialog. With this dialog you can perform actions like Undo and Redo, also it has a toolbar that has included some basic buttons like New, Save, Open, Copy, Cut, Paste, Tools, Help and allow user to add its own buttons and hide what doesn&#39;t need. This is a powerful dialog that is used by lots of specific dialogs in this project.
