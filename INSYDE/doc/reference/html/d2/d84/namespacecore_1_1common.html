◆ getCursor()

QCursor core::common::getCursor ( const QPoint &  pos,
const QRect &  rect,
int  clearance,
const QCursor &  defCursor = QCursor(Qt::ArrowCursor),
bool  ignoreCorners = false 
)

This utility function return a cursor depending on pos position, for example, when mouse is near to the borders of rect. Clearance determines the sensibility (minimum distance).

The cursors are resize arrows which depends on the side the pos parameter is.

This function is very useful when user wants to resize some rectangle or any resizable object.

Parameters
posThe position of the cursor.
rectThe rectangle to be processed.
clearanceThe minimum distance to update the cursor.
defCursorThe default cursor to be shown in case of pos is far away.
ignoreCornersIf true, the corners won't be taken in account.
Returns
Returns a QCursor object that depends on the pos parameter.
